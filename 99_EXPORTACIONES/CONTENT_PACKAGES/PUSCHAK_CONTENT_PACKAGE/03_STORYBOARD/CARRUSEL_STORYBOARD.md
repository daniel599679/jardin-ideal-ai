## LÓGICA DEL CARRUSEL

El carrusel Meta Ads funciona distinto al reel: el usuario desliza activamente.
Cada slide debe tener una razón para seguir deslizando.
La narrativa: **problema → proceso → resultado → prueba → acción.**

```
SLIDE 1 — HOOK / AVANT       → Provoca. "¿Así estaba?"
SLIDE 2 — PROCESSUS          → Credibilidad. "Así trabajamos."
SLIDE 3 — RÉVÉLATION [HERO]  → Deseo. "Así quedó."
SLIDE 4 — DÉTAIL             → Confianza. "Esta es la calidad."
SLIDE 5 — CTA                → Acción. "Esto puede ser tuyo."
```
