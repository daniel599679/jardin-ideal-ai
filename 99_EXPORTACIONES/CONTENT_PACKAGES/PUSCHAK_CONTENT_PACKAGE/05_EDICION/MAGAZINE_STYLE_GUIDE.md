# MAGAZINE STYLE GUIDE — JARDÍN IDEAL
**Aplicar a todo el contenido visual del paquete Puschak**

---

## PALETA DE COLOR

```
PAVÉ GRIS      : Gris pierre chaud  — référence Murano Gris Techo-Bloc
CLÔTURE        : Anthracite profond  #2C2C2C avec reflets satiné
VÉGÉTATION     : Vert émeraude      HSL +20 saturation sur base naturelle
CIEL           : Bleu cobalt        #1A6BAF — ni cyan, ni trop foncé
BRIQUE         : Rouge désaturé     -20 saturation pour ne pas dominer
OMBRES         : Bleu-violet doux   — jamais noires pures (max 65% densité)
LUMIÈRES       : Blanc chaud crémeux — température 5500–5800K
```

## TIPOGRAFÍA

```
DISPLAY (títulos, taglines):
  → Playfair Display Bold Italic
  → Georgia Italic (alternativa sistema)
  Uso: textos de impacto — "Votre entrée mérite mieux."

CUERPO (descripción, detalles):
  → Montserrat Regular / Light / Bold
  Uso: textos explicativos, listas, CTAs secundarios

TÉCNICO (teléfono, datos):
  → Montserrat Bold
  → SIEMPRE en dorado #C8A45A cuando es el teléfono CTA
```

## MOOD VISUAL

```
PALABRAS QUE LO DESCRIBEN:
  Minéral · Propre · Vivant · Premium · Authentique · Aspirationnel

PALABRAS QUE NO DEBE EVOCAR:
  Artificial · Saturé · HDR · Vintage · Filtré · Retravaillé

EL TEST: "¿Esta imagen podría aparecer en el catálogo Techo-Bloc 2026
         sin desentonar?" → Si la respuesta es SÍ: la retouche está bien.
```

## REGLAS DE COMPOSICIÓN

```
ESPACIO NEGATIVO  : Las mejores fotos tienen áreas sin información.
                    No llenar el frame — dejar respirar la imagen.
LÍNEAS GUÍA       : Usar líneas del pavé para guiar el ojo.
REGLA DE TERCIOS  : El sujeto principal en los tercios — no en el centro.
PRIMER PLANO      : Siempre incluir un elemento en primer plano cercano
                    (el pavé, una planta, una sombra) — da profundidad.
FONDO             : Limpiar o desfocar. Nunca distractivo.
```

## REGLAS DE TEXTO EN VISUAL

```
MÁXIMO 5 palabras por texto simultáneo en el reel
MÁXIMO 2 elementos de texto por slide de carrusel
El teléfono 514-266-2504 SIEMPRE en dorado #C8A45A
El logo SIEMPRE en esquina inferior izquierda
NUNCA texto blanco sobre zona blanca — agregar overlay oscuro
NUNCA más de 3 tipografías distintas en el mismo visual
```
