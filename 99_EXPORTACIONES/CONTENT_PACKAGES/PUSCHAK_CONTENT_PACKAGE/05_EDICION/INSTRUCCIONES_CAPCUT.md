```

3. Descargar la música seleccionada de la librería de CapCut (ver abajo).

## PASO 4 — TRANSICIONES

```
Entre clips 1→2  : "Whip" (barrido horizontal rápido) — comunica energía
Entre clips 2→3  : Corte directo
Entre clips 3→4  : Corte directo al beat
Entre clips 4→5  : Match cut (el pavé en proceso → pavé terminado)
                   Si el match no es posible: dissolve 0.3s
Entre clips 5→6  : Dissolve suave 0.3s
Entre clips 6→7  : Corte directo
Entre clips 7→8  : Dissolve suave
Entre clips 8→9  : Fade to black parcial (darken) 0.5s
```

**Regla:** No usar la misma transición más de 2 veces seguidas.
No usar "Zoom" ni "Spin" — son transiciones de baja calidad visual.

## PASO 7 — COLOR GRADE GLOBAL

```

  Seleccionar: "Warm" o "Natural" (no usar filtros de alto contraste ni vintage)

Ajustes manuales adicionales:
  Brillo      : +5
  Contraste   : +10
  Saturación  : +8
  Temperatura : +15 (más cálido — luz de verano)
  Nitidez     : +10

Nota: Estos ajustes aplican globalmente. Si algún clip se ve mal,
      ajustar ese clip individualmente con los mismos controles.
