# HOOKS — PROJET PUSCHAK
**Uso:** Primeros 3 segundos del reel + Texto de apertura en carrusel + Headline Meta Ads

Los hooks son la única variable que determina si el contenido se ve o no.
Todo lo demás es irrelevante si el hook falla.

---

## HOOKS PARA REEL (texto en pantalla — máximo 7 palabras)

### CATEGORÍA 1 — PREGUNTA QUE DUELE
```
H01  "Et si votre cour ressemblait à ça ?"
H02  "Combien d'étés avez-vous perdu dehors ?"
H03  "Votre cour vous embarrasse encore ?"
H04  "Vous avez du béton fissuré devant chez vous ?"
```

### CATEGORÍA 2 — CONTRASTE AVANT/APRÈS
```
H05  "5 jours. Une transformation complète."
H06  "Avant : béton fissuré. Après : regardez."
H07  "Ce n'est pas la même maison."
H08  "On a tout changé en une semaine."
```

### CATEGORÍA 3 — AUTORIDAD / PRUEBA SOCIAL
```
H09  "Ce qu'on a fait pour la famille Puschak à Montréal."
H10  "200 projets. Le même niveau d'exigence."
H11  "C'est pour ça qu'on est la référence à Montréal."
```

### CATEGORÍA 4 — URGENCIA DE TEMPORADA
```
H12  "Cet été, il reste quelques projets disponibles."
H13  "Nos disponibilités de juillet se remplissent."
H14  "Dernière chance de profiter de votre été."
```

### CATEGORÍA 5 — PROVOCACIÓN ASPIRACIONAL
```
H15  "Vos voisins vont remarquer la différence."
H16  "L'entrée de vos rêves existe vraiment."
H17  "Votre maison mérite mieux que du béton."
```

## MATRIZ DE SELECCIÓN — ¿Qué hook usar dónde?

| Formato | Hook recomendado | Alternativa |
|---|---|---|
| Reel (orgánico) | H01 — Pregunta que duele | H05 — Contraste |
| Reel (ad pagado) | H05 — 5 jours | H09 — Autoridad |
| Carrusel (ad) | "Glissez pour voir la transformation" | H06 — Contraste |
| Meta Ads hero | "Votre cour transformée en 5 jours" | "Votre entrée mérite mieux" |
| Stories | H12 o H13 — Urgencia | H15 — Aspiracional |

## REGLA DE ORO DEL HOOK

```
Un hook funciona si responde SÍ a las 3 preguntas:

1. ¿El espectador se identifica con el problema en menos de 2 segundos?
2. ¿Genera una pregunta en la mente ("¿Y entonces qué?") que obliga a seguir?
3. ¿Funciona sin sonido? (60% de los videos se ven en mute)

Si la respuesta a cualquiera de las 3 es NO → cambiar el hook.
```
