## TEXTOS QUE NUNCA DEBEN APARECER

| Texto prohibido | Razón |
|-----------------|-------|
| "Pré-approuvé" | No existe evaluación real |
| "Garantie X ans" para piscine, terrasse, clôture, etc. | No confirmado por Daniel |
| Precios exactos inventados (ex: "à partir de $4,500") | No confirmados |
| "Approuvé par Desjardins" o nombre de banco | No confirmado |
| "Meilleur prix garanti" | Claim sin respaldo |
| Inglés mezclado ("Landscaping experts") | LP y contenido es en francés |
