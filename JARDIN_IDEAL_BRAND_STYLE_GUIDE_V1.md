## 02 — COLOR GRADING OFFICIEL

### Le Look Jardín Ideal

```
MOT DIRECTEUR : Lumière naturelle maîtrisée.
                Pas de HDR. Pas de filtre vintage. Pas de saturation agressive.
                Une photo de Jardín Ideal doit sembler "vraie mais améliorée."
```

### Paramètres Lightroom Globaux — Tous Services

```
LUMIÈRE
  Exposition        : 0 à +0.4 (jamais surexposé — jamais burned)
  Récupération HL   : −30 à −60 (récupérer les blancs brûlés)
  Lumières          : −20 à −40
  Ombres            : +15 à +30 (ouvrir les ombres sans les aplatir)
  Noirs             : −10 à −20 (profondeur sans bouchage)
  Blancs            : +10 à +20

PRÉSENCE
  Clarté            : +8 à +15 (textures du matériau — jamais +25 et plus)
  Débruitage        : 0 (les textures sont la valeur ajoutée)
  Vibrance          : +10 à +18
  Saturation        : −5 à 0 (jamais augmenter la saturation globale)

COURBE DE TONS — Forme en S Légère
  Ombres            : +10
  Tons sombres      : 0
  Tons clairs       : 0
  Hautes lumières   : −10
  Point noir        : 15 (ne pas commencer à 0 — look cinématique)
  Point blanc       : 240 (ne pas aller à 255 — éviter le brûlé)
```

### HSL — Couleurs Spécifiques

```
ROUGE (brique, cheminée)
  Teinte      : 0  Saturation : −20  Luminosité : 0
  Raison: La brique trop rouge domine. On la neutralise.

ORANGE (terre, bois chaud)
  Teinte      : +5  Saturation : −10  Luminosité : +5
  Raison: Warmth sans agressivité.

JAUNE (lumière soleil, feuilles mortes)
  Teinte      : −5  Saturation : −15  Luminosité : +10
  Raison: Éviter le jaune criard dans les zones lumineuses.

VERT (végétation — LA COULEUR CLÉ)
  Teinte      : −8 à −12  Saturation : +15 à +22  Luminosité : +8 à +12
  Raison: Les verts naturels tirent vers le jaune-vert. On les pousse
          vers l'émeraude pour correspondre à l'identité de marque.

BLEU (ciel)
  Teinte      : +5 à +10  Saturation : +20 à +35  Luminosité : −5 à −10
  Raison: Pousser le ciel vers le bleu cobalt (#1A6BAF). Jamais cyan.

GRIS (pavé, béton — LA TEXTURE CLÉ)
  Pas de modification HSL — le gris se contrôle via clarté et courbes.
```

### Référence Sky Replacement

```
CIEL IDÉAL JARDÍN IDEAL
  Couleur     : Bleu cobalt moyen (#1A6BAF — ni trop clair, ni marine)
  Nuages      : 0 à 20% — quelques nuages blancs = naturel
  Horizon     : Transition douce (gradient de 10%)
  INTERDITS   : Ciel violet/rose (sunset dramatique), ciel bleu électrique,
                nuages d'orage, ciel de nuit

QUAND REMPLACER LE CIEL
  Ciel gris ou blanc brûlé → Toujours remplacer
  Ciel bleu mais plat → Remplacer si le projet le mérite
  Ciel parfait → Conserver l'original

OUTILS APPROUVÉS
  Lightroom : Remplacement de ciel automatique
  Photoshop : Sky Replacement (Édition → Remplacement du ciel)
  Luminar   : AI Sky Replacement
```

### Paramètres Vidéo — Reels

```
CORRECTION COLORIMÉTRIQUE DE BASE (CapCut ou Premiere)
  Luminosité    : +5 à +10
  Contraste     : +15 à +25
  Saturation    : +10 à +15
  Température   : 5400–5600K (légèrement chaud — jamais froid)
  Teinte        : 0

LUT RECOMMANDÉ (si disponible dans CapCut)
  "Cinematic Warm" ou équivalent léger
  Intensité: 30–40% (jamais 100% — trop artificiel)

COHÉRENCE VIDÉO/PHOTO
  La vidéo doit sembler appartenir au même univers que les photos.
  Si la photo est retouchée avec +15 vibrance, la vidéo doit l'être aussi.
```

```
HIÉRARCHIE PAR PIÈCE
  Niveau 1 (Hook / Titre)  : Playfair Display Bold Italic — plus grande taille
  Niveau 2 (Bénéfice)      : Montserrat SemiBold — 60% de la taille du titre
  Niveau 3 (CTA / Détail)  : Montserrat Regular — 50% de la taille du titre

  Interlignage   : 1.2 à 1.3 (jamais serré, jamais trop aéré)

  Hooks et titres   : Casse mixte (majuscule première lettre seulement)

  Reel frame       : Maximum 5 mots par élément de texte visible simultanément
  Slide carrusel   : Maximum 2 éléments de texte par slide
  Image statique   : Maximum 3 éléments de texte (titre + sous-titre + CTA)
  JAMAIS            : 3 typographies différentes dans le même visuel
```

## 04 — STYLE HERO IMAGE

### Définition du Hero Jardín Ideal

```
Le hero est la seule photo que le client verra peut-être.
Il doit répondre à une seule question en 1.5 secondes:
"Est-ce que je veux que mon espace ressemble à ça ?"
Si la réponse n'est pas instinctivement OUI — ce n'est pas le hero.
```

### Critères Visuels du Hero

```
COMPOSITION
  · La règle des tiers est respectée (sujet sur les intersections)
  · Il y a un élément en premier plan proche (le matériau, une plante, une ombre)
  · L'espace "respire" — 20–30% de l'image sans information
  · Une ligne directrice (joint de pavé, bordure, perspective) guide le regard
  · Le ciel visible = maximum 25% du frame (pas de photo "ciel dominant")

LUMIÈRE
  · Lumière naturelle exclusivement (jamais de flash)
  · Heure idéale: 07:00–10:00 ou 15:00–18:00 (lumière rasante)
  · Pas de photo à midi (lumière zénithale = ombres dures, sans profondeur)
  · Les ombres portées sont bienvenues — elles donnent la profondeur et l'échelle

COULEURS
  · Les verts de la végétation sont saturés vers l'émeraude (HSL appliqué)
  · Le ciel est bleu cobalt (pas gris, pas blanc, pas cyan)
  · Le matériau principal (pavé, béton) garde sa couleur naturelle
  · Aucune dominante colorée artificielle sur toute l'image

PROPRETÉ
  · Zéro câble électrique visible
  · Zéro outil ou matériau de construction
  · Zéro véhicule en premier plan
  · Zéro personne (sauf décision créative délibérée pour lifestyle)
  · Le site est nettoyé avant la prise de vue

FORMATS D'EXPORT
  META 4:5         → 1080 × 1350 px — annonçe principale
  STORY 9:16       → 1080 × 1920 px — Instagram/Facebook Story
  CARRÉ 1:1        → 1080 × 1080 px — feed carré / post organique
  WEB 16:9         → 1920 × 1080 px — site web / présentation
  Résolution       : 72 dpi (digital) — toujours sRGB (jamais Adobe RGB)
  Format fichier   : JPG qualité 90% (équilibre poids/qualité)
```

### Ce Qu'un Hero Jardín Ideal N'est Pas

```
✗  Une photo prise de l'intérieur du véhicule
✗  Une photo avec la pluie ou le sol mouillé
✗  Une photo en contre-jour dur (sujet en silhouette non intentionnel)
✗  Une photo sombre ou sous-exposée
✗  Une photo avec filtre Instagram (Clarendon, Juno, Valencia, etc.)
✗  Un panorama horizontal de 21:9 (ne s'adapte pas aux formats Meta)
✗  Une photo avec le client visible sans son autorisation signée
```

## 05 — STYLE DES REELS

### L'ADN du Reel Jardín Ideal

```
Un reel Jardín Ideal n'est pas une publicité.
C'est une transformation racontée en moins de 40 secondes.
Le spectateur doit ressentir que ce n'est pas possible —
et que ça l'est. Pour lui. Dans sa rue.
```

### Structure Narrative Standard

```
PHASE 1 — LE PROBLÈME (0–20% de la durée)
  La situation avant, le terrain difficile, l'espace abandonné.
  Montrer sans commenter — l'image suffit.
  Texte optionnel: "Avant" ou "Il y a [X] semaines"

PHASE 2 — LA RÉVÉLATION (20–30% de la durée)
  Le coupé direct vers l'après. Pas de transition douce — cut brut.
  Le contraste doit être maximal. C'est le moment le plus important du reel.

PHASE 3 — LA DÉMONSTRATION (30–70% de la durée)
  Le process. Le savoir-faire. L'équipe au travail.
  Si possible: de vraies images du chantier.
  Sinon: série de photos avec Ken Burns effect (mouvement lent).

PHASE 4 — LE RÉSULTAT (70–90% de la durée)
  Le recorrido lent. La caméra se promène dans l'espace fini.
  Pas de rush. Le spectateur doit s'imaginer chez lui.

PHASE 5 — L'APPEL (90–100% de la durée)
  Fond vert profond #052B16.
  Texte: Le hook de clôture + téléphone en or.
  Durée: 3–4 secondes — assez pour lire, pas trop pour ennuyer.
```

### Paramètres Techniques

```
FORMAT          : 9:16 vertical (1080 × 1920 px)
DURÉE           : 20–45 secondes selon le service (voir COS)
FRÉQUENCE       : 30 fps (pas 60 — trop "télévisuel", pas assez "cinématique")
CODEC EXPORT    : H.264 MP4 — qualité haute
BITRATE         : minimum 10 Mbps pour éviter la compression Meta
AUDIO           : −14 LUFS intégrés (standard Meta — évite la saturation)
SOUS-TITRES     : Non (les textes à l'écran font office de sous-titres)
```

### Règles de Montage

```
RYTHME
  Phase 1 (avant)   : Coupes lentes — 3–5 secondes par plan
  Phase 2 (révél.)  : Cut brutal instantané — pas de transition
  Phase 3 (process) : Coupes rapides — 1.5–2.5 secondes par plan
  Phase 4 (résultat): Coupes lentes — 2–4 secondes par plan
  Phase 5 (CTA)     : Plan fixe de 3–4 secondes

SPEED RAMPING (si disponible dans CapCut)
  Accélération légère (1.5×) pendant le processus
  Retour à 1× pour le résultat
  JAMAIS de slow motion gratuit — seulement si le moment le justifie

TEXTE EN MOUVEMENT
  Entrée         : Fade in ou slide depuis le bas (jamais de pop ou de bounce)
  Sortie         : Fade out ou disparition sur le cut
  Position       : Tiers inférieur de l'écran (ne pas masquer le sujet)
  JAMAIS         : Texte qui tourne, qui tremble, qui clignote

FRAME RATE DU TEXTE
  Le texte apparaît sur une coupe — jamais à mi-chemin entre deux plans
```

## 06 — MUSIQUE RECOMMANDÉE

### Le Son Jardín Ideal

```
MOOD GÉNÉRAL   : Cinématique · Inspirant · Progressif · Chaud
                 "La maison de vos rêves se construit devant vos yeux"

PAS ÇA         : Electro agressif · Hip-hop · Musique "virale TikTok" ·
                 Ballades romantiques · Musique latine/dansante ·
```

```
REELS 20–25 SECONDES (Balcons, Escaliers, Clôtures)
  Style    : Piano ambiant + percussions légères qui montent
  Énergie  : Basse au début, pic à la révélation, résolution calme

REELS 35–45 SECONDES (Cour Arrière, Piscines, Rénovation intérieure)
  Style    : Orchestral avec cordes + production moderne
  Énergie  : Tension légère en phase 1, explosion à la révélation, résolution épique

```

```

                          (pas de silence au début)

BEAT-MATCHING           : Si possible, les coupes coïncident avec les beats
                          de la musique (CapCut peut détecter automatiquement)
```

## 07 — STYLE DES TRANSITIONS

### Règle Absolue

```
La transition la plus puissante est l'absence de transition.
Un cut brut au bon moment est toujours plus fort qu'un effet élaboré.
Les effets de transition ne compensent pas un mauvais montage.
```

### Transitions Approuvées par Contexte

```
CUT BRUT (priorité absolue)
  Quand        : Avant → Après (la révélation). Toujours.
  Effet        : Instantané. Pas de frame intermédiaire.
  Raison       : Le contraste maximum crée l'impact maximum.
                 Une transition adoucie dilue l'effet "wow".

FADE IN/OUT (fond noir ou fond blanc)
  Quand        : Début et fin du reel uniquement.
  Durée        : 0.3–0.5 secondes. Pas plus.
  Fond         : Noir pour les reels dramatiques. Blanc pour les contenus lifestyle.

GLISSEMENT HORIZONTAL DOUX (slide)
  Quand        : Entre deux plans du même espace (phase 4 — résultat)
  Durée        : 0.4–0.6 secondes.
  Direction    : Toujours dans le même sens dans un reel (pas d'inversion)
  JAMAIS        : Comme transition avant → après

ZOOM PROGRESSIF (Ken Burns)
  Quand        : Sur les photos fixes (quand il n'y a pas de vidéo)
  Mouvement    : Zoom in très lent (1× → 1.05× sur 3–4 secondes)
  Raison       : Donne de la vie à une image fixe sans paraître artificiel
```

### Transitions Interdites

```
✗  Wipe (balayage) — trop daté
✗  Rotation ou flip 3D — amateur
✗  Glitch effect — incompatible avec le positionnement premium
✗  Effets de particules (étincelles, confettis) — trop agressif
✗  Morphing automatique CapCut — trop technologique, pas naturel
✗  Transition "match cut" (bonne technique — seulement si planifiée au tournage)
```

## 08 — STYLE DES MINIATURES

### Le Principe de la Miniature

```
La miniature est vue en 60px × 60px dans un feed saturé.
Elle doit être lisible et désirable à cette taille.
Si on ne peut pas deviner le "wow" en miniature — refaire la miniature.
```

### Anatomie de la Miniature Jardín Ideal

```
ZONE 1 — L'IMAGE (70–80% de la surface)
  · La photo hero retouchée ou le frame "révélation" du reel
  · Contraste élevé — pas de photo sombre ou floue
  · L'élément principal (pavé, piscine, cour) doit être identifiable
    même à petite taille

ZONE 2 — LE TEXTE (20–30% de la surface, toujours en bas ou en haut)
  · 1 à 3 mots maximum en Playfair Display Bold Italic
  · Blanc (#FFFFFF) avec ombre noire légère (opacity 60%, blur 4px)
  · Jamais plus de 5 mots visibles en miniature

ZONE 3 — L'ACCENT DE MARQUE (optionnel)
  · Un liseré vert profond #052B16 de 4px sur un bord
  · OU un badge "AVANT / APRÈS" en Montserrat Bold blanc sur #008E3F
```

### Templates de Miniatures par Type de Contenu

```
MINIATURE AVANT/APRÈS (la plus performante)
  · Image divisée verticalement: gauche = avant, droite = après
  · Ligne de séparation blanche de 3px au centre
  · Text overlay: "AVANT" (gauche, petit) et "APRÈS" (droite, petit)
  · Le contraste doit être maximal — utiliser le pire avant et le meilleur après

MINIATURE RÉSULTAT PUR
  · La photo hero en plein cadre
  · Dégradé du bas vers le haut: #052B16 transparent → opaque (30% de la hauteur)
  · Le texte du service en bas dans le dégradé

MINIATURE "CHIFFRE"
  · Photo hero en fond
  · Badge central ou coin: overlay #052B16 avec chiffre en blanc
    "[X] m²" ou "[X] jours" ou "50 ans"
  · Fonctionne bien pour Pavé-Uni et Cour Avant
```

### Règles de la Miniature

```
TAILLE CIBLE         : 1080 × 1080 px (carré pour la polyvalence)
FORMAT               : JPG qualité 85%
TEST OBLIGATOIRE     : Réduire l'aperçu à 10% de la taille et vérifier
                       que l'image reste lisible et attrayante
RÈGLE DE CONTRASTE   : WCAG AA minimum — le texte doit être lisible
                       (ratio de contraste ≥ 4.5:1)
```

## 09 — STYLE DES CTA

### Le CTA Jardín Ideal

```
Un CTA Jardín Ideal ne supplie pas.
Il invite avec confiance.
"Demandez votre soumission" — pas "Contactez-nous maintenant !!!"
Le ton est celui d'un professionnel qui sait que son travail parle pour lui.
```

### Hiérarchie des CTAs

```
NIVEAU 1 — CTA PRINCIPAL (un seul par pièce de contenu)
  Version longue  : "Demandez votre soumission gratuite"
  Version courte  : "Soumission gratuite"
```

## DÉCISIONS D'EXCEPTION

*Quand une situation sort du cadre de cette guide:*

```
RÈGLE D'EXCEPTION
  Toute décision qui déroge à ce document doit être:
  1. Justifiée par un objectif créatif spécifique (pas par la commodité)
  2. Documentée dans le fichier du projet concerné
  3. Approuvée par Daniel

EXEMPLES DE DÉROGATIONS ACCEPTABLES
  · Utiliser un coucher de soleil (orange/rose) si le projet est une piscine
    et que la photo est exceptionnelle et rare
  · Utiliser une personne dans le contenu pour un témoignage client signé
  · Utiliser une musique connue sur un reel organique non boosté
    si elle correspond parfaitement à la narration

EXEMPLES DE DÉROGATIONS INACCEPTABLES
  · Changer le téléphone de couleur "pour cette fois"
  · Utiliser une photo de mauvaise qualité parce qu'on n'en a pas d'autre
  · Publier avec un ciel gris "parce que c'est urgent"
```
