# PROTOCOLO DE INICIO DE SESIÓN
**Para:** Claude Web / Claude Desktop / Claude Code / Manus / ChatGPT
**Objetivo:** Activar el contexto completo del sistema Jardín Ideal en menos de 3 minutos.

---

## PASO 1 — LEER (obligatorio, en orden)

```
1. 10_MEMORIA_EMPRESARIAL/MEMORIA_GLOBAL.md
   → Contexto completo: empresa, sistemas, reglas, equipo

2. 10_MEMORIA_EMPRESARIAL/ESTADO_ACTUAL_ECOSISTEMA.md
   → Qué está completo, qué está en progreso, qué está pendiente

3. 10_MEMORIA_EMPRESARIAL/PROXIMOS_PASOS.md
   → Prioridades de esta semana y de hoy
```

---

## PASO 2 — VERIFICAR EMPRESA ACTIVA

Leer `01_EMPRESAS/EMPRESA_ACTIVA.md`

```
Si EMPRESA_ACTIVA = JARDIN_IDEAL → cargar perfil de Jardín Ideal
Si EMPRESA_ACTIVA = GROUPE_IDEAL → cargar perfil de Groupe Ideal
```

---

## PASO 3 — LEER SEGÚN EL TIPO DE SESIÓN

### Sesión de VENTAS / CRM
```
→ 10_MEMORIA_EMPRESARIAL/SISTEMA_CLIENTE_IDEAL.md
→ JARDIN_IDEAL_IDEAL_CUSTOMER_SYSTEM.md (si necesitas scripts detallados)
→ 06_CRM/ (pipeline activo)
```

### Sesión de CONTENIDO / MARKETING
```
→ 10_MEMORIA_EMPRESARIAL/SISTEMA_CONTENIDO_FLOAT.md
→ CONTENT_OPERATING_SYSTEM.md (protocolo completo por servicio)
→ JARDIN_IDEAL_BRAND_STYLE_GUIDE_V1.md (identidad visual)
→ 05_MARKETING/ (campañas activas)
```

### Sesión de OPERACIONES
```
→ 01_AGENTES/agente_operaciones.md
→ 03_CALENDARIO/ (agenda del día)
→ 08_REPORTES/ (métricas recientes)
```

### Sesión de SISTEMA / INFRAESTRUCTURA
```
→ ECOSISTEMA_MAESTRO.md
→ 10_MEMORIA_EMPRESARIAL/DECISIONES_ESTRATEGICAS.md
→ 10_MEMORIA_EMPRESARIAL/APRENDIZAJES_Y_ERRORES.md
→ 07_AUTOMATIZACIONES/
```

### Sesión de CONTENIDO AUDIOVISUAL (Manus / video)
```
→ 10_MANUS/guia_manus.md
→ 10_MANUS/sesion_diaria.md
→ 05_MARKETING/AGENTES_CALIDAD_Y_VIDEO/
```

---

## PASO 4 — VERIFICACIÓN DE CONTEXTO (30 segundos)

Antes de empezar a trabajar, confirmar mentalmente:

```
☐ ¿Sé qué empresa está activa?
☐ ¿Conozco el estado de los proyectos activos?
☐ ¿Sé cuáles son las prioridades de hoy?
☐ ¿Tengo claro el idioma de trabajo? (Interno: español / Clientes: francés)
☐ ¿Recuerdo la reunión de las 09:00 AM? (inamovible)
```

---

## PASO 5 — INICIO DE SESIÓN CON CLAUDE CODE

Al abrir una sesión en el proyecto, Claude Code ya carga el historial.
Si la sesión es nueva o continúa de otra, usar el resumen de contexto:

```
"Contexto activo: Jardín Ideal AI System.
 Empresa activa: Jardín Ideal.
 FLOAT V1 completo. ICS completo. Memoria unificada creada.
 Prioridad actual: [leer PROXIMOS_PASOS.md]"
```

---

## PARA MANUS

Al iniciar sesión en Manus, compartir el contenido de `MEMORIA_GLOBAL.md` como primer mensaje.
Luego cargar el archivo específico de la sesión desde `10_MANUS/`.

---

## PARA CLAUDE WEB / CHATGPT

Compartir como primer mensaje:
1. El contenido de `MEMORIA_GLOBAL.md`
2. El tipo de sesión y el objetivo específico

Ejemplo:
```
"Eres el asistente del sistema Jardín Ideal.
[pegar contenido de MEMORIA_GLOBAL.md]
Hoy trabajaremos en: [objetivo de la sesión]"
```

---

## TIEMPO ESTIMADO DE ONBOARDING

| Herramienta | Tiempo de setup |
|------------|----------------|
| Claude Code | 0 min (memoria persistente en proyecto) |
| Claude Desktop | 1–2 min (leer MEMORIA_GLOBAL.md) |
| Claude Web | 2–3 min (pegar contexto + tipo de sesión) |
| Manus | 2–3 min (pegar MEMORIA_GLOBAL.md + archivo sesión) |
| ChatGPT | 3–5 min (pegar contexto + instrucciones de sistema) |
