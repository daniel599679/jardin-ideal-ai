## QUÉ ES FLOAT

FLOAT convierte cada proyecto completado de Jardín Ideal en una máquina de generación de leads.
Un proyecto terminado produce: hero image + reel + carrusel + campaña Meta Ads en menos de 60 minutos de trabajo humano.

---

## PROTOCOLO DE 60 MINUTOS (por proyecto)

| Minuto | Tarea | Herramienta |
|--------|-------|-------------|
| 0–10 | Selección del hero image (ángulo perfecto, luz, composición) | Lightroom / galería |
| 10–30 | Retoque en Lightroom (parámetros exactos del Brand Guide) | Lightroom |
| 30–45 | Copy del hook bank — adaptar a variables del proyecto | ICS / COS |
| 45–60 | Configurar Meta Ads (segmentación, presupuesto, formatos) | Meta Ads Manager |

## IDENTIDAD VISUAL — PARÁMETROS CLAVE

### Paleta
```
#052B16  Verde oscuro   — fondos, overlays de texto
#008E3F  Verde claro    — acento, botones, iconos
#C8A45A  Dorado         — CTA teléfono SOLO
#F5F2EB  Crema          — fondos secundarios, textos sobre oscuro
#FFFFFF  Blanco         — texto sobre colores oscuros
```

### Lightroom — Ajustes Globales
```
Exposición:    0 a +0.4
Sombras:       +15 a +30
Claridad:      +8 a +15
Vibrance:      +10 a +18
Saturación:    -5 a 0
```

### Lightroom — HSL
```
Verde (Hue):         -5 a +5
Verde (Saturación):  +15 a +22
Azul cielo (Sat):    +20 a +35
```

### Reel
```
Duración:    15–30 seg (ideal 20 seg)
Formato:     9:16 vertical (Instagram/TikTok) + 1:1 (Feed)
FPS:         30
Codec:       H.264 (MP4)
Audio:       -14 LUFS (estándar Instagram)
Estructura:  Problema → Revelación → Demostración → Resultado → CTA
```

### Tipografía
```
Display/hooks:  Playfair Display Bold Italic
Cuerpo/CTA:     Montserrat Regular + SemiBold
```

## CÓMO ACTIVAR FLOAT PARA UN NUEVO PROYECTO

```
1. Proyecto completado → fotografías en cámara/celular
2. Abrir CONTENT_OPERATING_SYSTEM.md → sección del servicio
3. Seleccionar hero según los ángulos recomendados
4. Aplicar parámetros Lightroom del Brand Style Guide
5. Seleccionar hook del banco → adaptar variables [VILLE], [SUPERFICIE], etc.
6. Publicar en Instagram/Facebook
7. Lanzar Meta Ad con la creatividad principal
8. Registrar en KPI de Producción (FLOAT V2 Módulo 7 cuando esté listo)
```
