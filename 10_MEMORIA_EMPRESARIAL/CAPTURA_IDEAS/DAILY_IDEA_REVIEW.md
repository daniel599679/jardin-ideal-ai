# DAILY IDEA REVIEW
**Revisión diaria de ideas y priorización**
**Versión:** 1.0 | **Fecha:** 2026-06-23
**Duración:** 15 minutos máximo (parte del START_DAY_PROTOCOL)
**Cuándo:** Cada mañana entre 07:30 y 08:45 (antes de la reunión de las 09:00)

---

## POR QUÉ EXISTE ESTE RITUAL

Sin una revisión diaria, las ideas se acumulan en el inbox sin ejecutarse.
Sin ejecutarse, el sistema no evoluciona.
Sin evolución, el objetivo de 7M CAD se vuelve más difícil.

**15 minutos al día de revisión de ideas = el sistema mejora continuamente.**

---

## CHECKLIST DIARIO — SECUENCIA EXACTA

### 07:30 — REVISIÓN DEL INBOX (5 min)

```
☐ Abrir IDEA_INBOX.md
☐ ¿Hay ideas nuevas marcadas como 🆕 NUEVA?
  SÍ → para cada una:
    → Identificar a qué sistema(s) conecta (FLOAT / Marketing / Ventas / Automatizaciones / ICS / Ops)
    → Decidir: P1 / P2 / P3 / BACKLOG / DESCARTAR
    → Mover el ID al nivel correspondiente en IDEAS_PRIORIZADAS.md
    → Cambiar el estado en IDEA_INBOX a "PROCESADA → [destino]"
  NO → continuar al siguiente paso

☐ ¿Hay notas de voz o mensajes WhatsApp de ayer marcados "IDEA:"?
  SÍ → transcribir y agregar al IDEA_INBOX con la plantilla
  NO → continuar

☐ ¿Hay ideas del BACKLOG que hoy sean más urgentes (por un evento, un cliente, etc.)?
  SÍ → promover al nivel de prioridad correcto
  NO → continuar
```

---

### 07:35 — REVISIÓN DE IDEAS_PRIORIZADAS (5 min)

```
P1 — GENERA INGRESOS:
☐ ¿Cuántas P1 hay activas? (máximo permitido: 3)
☐ ¿Alguna P1 está bloqueada? → documentar el bloqueo y mover a P2 si no se puede resolver hoy
☐ ¿Alguna P1 se completó ayer? → mover a COMPLETADAS con fecha y resultado
☐ La P1 más urgente de hoy: ________________

P2 — MEJORA PRODUCCIÓN:
☐ ¿Hay P2 que se puedan ejecutar en <2 horas hoy? → promover temporalmente a P1
☐ ¿Alguna P2 lleva >2 semanas sin moverse? → revisar si sigue siendo relevante

P3 — INFRAESTRUCTURA:
☐ Revisión rápida — ¿sigue siendo válida la lista? (30 segundos)
```

---

### 07:40 — DECISIÓN DEL DÍA (5 min)

```
Pregunta principal: ¿Cuál es la idea o tarea de mayor impacto que puedo ejecutar HOY?

Criterio de selección:
  1. ¿Hay una P1 lista para ejecutar (sin bloqueos)?
     SÍ → esa es la prioridad #1 del día
  2. ¿No hay P1 disponible? → ¿Hay una P2 ejecutable hoy?
     SÍ → ejecutar esa P2
  3. ¿Solo hay P3? → ¿Vale la pena empezar algo de infraestructura hoy?
     Depende del resto de la agenda

REGISTRO DE DECISIÓN DEL DÍA:
  Fecha: ________________
  Idea/Tarea principal: ________________
  Prioridad: P1 / P2 / P3
  Sistema que impacta: ________________
  Tiempo estimado: _____ minutos
  Herramienta principal: ChatGPT / Claude Web / Claude Desktop / Claude Code / Manus
```

## REVISIÓN SEMANAL DE IDEAS (lunes, 08:00 – 08:30)

El lunes se hace una revisión más profunda de 30 minutos:

```
LIMPIEZA DEL INBOX:
☐ ¿Hay ideas en el inbox de más de 7 días sin procesar? → procesar todas ahora
☐ ¿El inbox tiene más de 10 ideas activas? → señal de acumulación — hacer sesión de clasificación

REVISIÓN DE P1:
☐ ¿Las P1 actuales siguen siendo P1? → el mercado cambia, las prioridades también
☐ ¿Hay P1 del trimestre anterior sin completar? → son P2 — ajustar el nivel
☐ ¿Las P1 completadas generaron el resultado esperado? → documentar en IDEAS_COMPLETADAS

REVISIÓN DE P2:
☐ ¿Hay P2 que llevan >1 mes? → o las ejecutamos esta semana o las mandamos a P3
☐ ¿Hay P2 que se volvieron urgentes por algún cambio en el mercado? → promover a P1

REVISIÓN DE P3:
☐ ¿Hay P3 que ya deberían ser P2? → mover si la infraestructura es urgente
☐ ¿Hay P3 que ya no son relevantes? → mover al BACKLOG o descartar

NUEVO SPRINT:
☐ Definir las 3 ideas/tareas más importantes de esta semana
☐ Asignar a las herramientas correspondientes (ChatGPT / Claude / Manus)
☐ Bloquear tiempo en el calendario
```

## SEÑALES DE QUE EL SISTEMA FUNCIONA

```
✅ El inbox tiene menos de 5 ideas sin procesar
✅ Hay entre 1 y 3 ideas activas en P1
✅ Al menos 1 idea se completa por semana
✅ Las P3 avanzan aunque sea lentamente
✅ El proceso de revisión diaria toma <15 minutos consistentemente
```

## SEÑALES DE QUE EL SISTEMA ESTÁ EN PROBLEMA

```
🔴 El inbox tiene más de 20 ideas sin procesar → hacer sesión de limpieza urgente
🔴 No se completó ninguna P1 en 2 semanas → revisar si están bien priorizadas o si hay bloqueos
🔴 Hay más de 5 P1 simultáneas → racionalizar, dejar máximo 3
🔴 La revisión diaria toma más de 30 minutos → el sistema está en caos, re-ordenar
🔴 No se ha revisado IDEAS_PRIORIZADAS en más de 5 días → el sistema está abandonado
```
