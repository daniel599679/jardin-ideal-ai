## ESTRATEGIA DE REMARKETING

### Diferencia fundamental con el Hero Post

El Hero Post le habla a **alguien que no nos conoce**.
El Remarketing le habla a **alguien que ya nos vio pero no actuó**.

El error más común: usar el mismo copy. La misma imagen. El mismo ángulo.

Esta pieza usa una imagen DIFERENTE, una emoción DIFERENTE, y un copy DIFERENTE — porque el espectador ya sabe lo que hacemos. Ahora necesita el empuje final para actuar.

### Audiencia objetivo

```
Audiencia cálida (configurar en Meta Ads Manager):
  ✓ Personas que vieron el Hero Post (pero no hicieron clic)
  ✓ Personas que vieron ≥50% del Reel
  ✓ Visitantes del sitio jardin-ideal.com (últimos 30 días)
  ✓ Personas que interactuaron con el perfil de Instagram (últimos 60 días)
  
  Excluir:
  ✗ Personas que ya enviaron formulario (píxel)
  ✗ Clientes actuales (lista manual de emails)
```

## MOTIVO DE SELECCIÓN

1. **Visual diferente del hero** — El espectador ya vio las marches flottantes blancas. Esta imagen ofrece un ángulo completamente diferente (escalera espiral negra, urbano, dramático). Evita la fatiga creativa.

2. **Emoción diferente** — El hero activó aspiración. Esta imagen activa sofisticación urbana y sentido de urgencia más sutil.

3. **Contexto urbano Montreal** — Los escaliers de hierro negro son iconos de Montreal. El espectador se identifica inmediatamente con la arquitectura.

4. **Ángulo dramático** — El ángulo bajo mirando hacia arriba comunica "este trabajo tiene carácter". Para una audiencia cálida que ya sabe lo que hacemos, esto consolida la autoridad.

---

## EMOCIÓN QUE TRANSMITE

**Principal:** Sofisticación urbana — "Esto tiene clase"
**Secundaria:** Urgencia suave — "No voy a seguir posponiendo esto"
**Terciaria:** Pertinencia — "Esto es para propiedades como la mía"
