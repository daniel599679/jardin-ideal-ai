# DAILY CONTENT QUEUE
**Cola de Producción Diaria — Jardín Ideal**
**Versión:** 1.0 | **Fecha:** 2026-06-23
**Integración:** FLOAT V2 Premium
**Frecuencia:** Generar cada mañana a las 07:30 AM, antes de la reunión de las 09:00

---

## QUÉ ES ESTE DOCUMENTO

La Cola de Producción Diaria define exactamente qué 5 piezas de contenido se producen cada día. No es una lista de ideas — es una lista de órdenes de producción con formato, assets asignados y deadline.

**Principio:** Cada día se producen las 5 piezas, o se registra por qué no (falta de material, aprobación pendiente, etc.).

---

## LAS 5 PIEZAS DIARIAS OBLIGATORIAS

### PIEZA 1 — HERO POST

```
PROPÓSITO: Construir percepción de lujo y autoridad de marca
CANAL: Instagram Feed + Facebook Feed (publicación orgánica permanente)
FORMATO: 1:1 o 4:5 (feed permanente)
FLOAT V2: Template A "La Portada" (imagen hero dominant)

REQUISITOS:
  ✓ VISUAL_SCORE ≥ 90/100 (PREMIUM_VISUAL_SCORE)
  ✓ Hero image siguiendo HERO_IMAGE_FACTORY (hora dorada, set limpio)
  ✓ Hook ≤ 8 palabras en Playfair Display Bold Italic
  ✓ Logo esquina inferior derecha
  ✓ 0% espacio de texto agresivo — imagen habla sola

TIEMPO DE PRODUCCIÓN: 20–35 minutos

FORMATO DE LA ORDEN:
  PROYECTO: ________________
  ASSET: [nombre del archivo hero]
  VISUAL_SCORE: ___ / 100
  HOOK: ________________
  SERVICIO: ________________
  CAPTION (francés): ________________
  HASHTAGS: ________________
  HORA DE PUBLICACIÓN: ________
  ESTADO: PENDIENTE / EN PRODUCCIÓN / APROBADO / PROGRAMADO

SI NO HAY MATERIAL ≥90: documentar → "HERO PENDIENTE — esperar nueva sesión fotográfica"
```

### PIEZA 2 — REEL EMOCIONAL

```
PROPÓSITO: Alcance masivo, engagement, generación de leads orgánicos
CANAL: Instagram Reels + Facebook Reels
FORMATO: 9:16 vertical, 20–45 segundos
FLOAT V2: Arquitectura 5 fases (EMOTIONAL_REEL_ENGINE)

ARQUITECTURA OBLIGATORIA:
  0–3s   HOOK        → imagen o texto que detiene el scroll
  3–6s   PROBLEMA    → contexto del antes / del dolor
  6–14s  REVELACIÓN  → 80% del presupuesto visual aquí (HERO image ≥90pts)
  14–18s PRUEBA SOCIAL → dato concreto o testimonio implícito
  18–fin CTA          → número en dorado, una sola acción

MÚSICA: seleccionar de MUSIC_LIBRARY_SYSTEM según emoción objetivo
  Transformación → REVELACIÓN (70–90 BPM)
  Aspiración → ASPIRACIÓN (80–100 BPM)
  Proceso → CONFIANZA (100–120 BPM)
LUFS: -14 LUFS obligatorio

TIEMPO DE PRODUCCIÓN: 45–90 minutos (CapCut o Manus)

FORMATO DE LA ORDEN:
  PROYECTO: ________________
  TIPO DE REEL: AVANT/APRÈS / PROCESO / PORTFOLIO / TESTIMONIO
  ASSETS: [lista de archivos en orden]
  HERO FRAME: [archivo del mejor asset] SCORE: ___ / 100
  EMOCIÓN OBJETIVO: ________________
  MÚSICA ELEGIDA: ________________ (fuente: Epidemic/Artlist/Musicbed)
  DURACIÓN OBJETIVO: ___ seg
  HOOK VISUAL: ________________
  TEXTO OVERLAY (escena por escena): ________________
  ESTADO: PENDIENTE / EN PRODUCCIÓN / REVISIÓN HUMANA / APROBADO / PROGRAMADO

SI NO HAY MATERIAL SUFICIENTE: documentar → "REEL PENDIENTE — [razón]"
```

### PIEZA 3 — STORY

```
PROPÓSITO: Mantener visibilidad diaria con audiencia existente; puente a otras piezas
CANAL: Instagram Stories + Facebook Stories (efímero, 24h)
FORMATO: 9:16 vertical, imagen estática o video ≤15 segundos
FLOAT V2: Template B "El Overlay Elegante" para imágenes; Emotional Reel Engine (primeros 15s) para video

TIPOS ROTATIVOS (elegir según el día):
  Lunes: Teaser del hero post de la semana → "Swipe up para ver el proyecto completo"
  Martes: Pregunta educativa con encuesta ("Savez-vous combien dure le pavé-uni?")
  Miércoles: Clip de proceso de obra (15s) → genera curiosidad antes del reel
  Jueves: Cita textual del cliente sobre fondo verde oscuro
  Viernes: "Slots disponibles" o CTA directo
  Sábado: Behind the scenes — foto casual del equipo
  Domingo: Inspiración visual + mensaje de la semana

TEXTO EN STORY: máximo 5 palabras visibles + número de teléfono si hay CTA

TIEMPO DE PRODUCCIÓN: 10–15 minutos

FORMATO DE LA ORDEN:
  TIPO DE STORY: ________________
  ASSET: [archivo o texto si es solo texto/encuesta]
  VISUAL_SCORE (si es imagen): ___ / 100 (puede ser ≥75 para stories)
  TEXTO: ________________
  DURACIÓN: ___ segundos
  LINK/SWIPE UP: ________________ (si aplica)
  STICKER: Encuesta / Cuenta regresiva / Link / Ninguno
  ESTADO: PENDIENTE / LISTO / PROGRAMADO
```

### PIEZA 4 — POST EDUCATIVO

```
PROPÓSITO: Construir autoridad, diferenciarse de competidores, generar confianza pre-compra
CANAL: Instagram Feed + Facebook + Google Business
FORMATO: Carrusel (5–7 slides) o imagen única con caption largo
FLOAT V2: Template D "El Carrusel Editorial" + MAGAZINE_EDITORIAL_STANDARD (copy)

TEMAS POR ROTACIÓN:
  Semana A: Material (pavé-uni, composite, béton, pierre)
  Semana B: Proceso (cómo se instala, cuánto tarda, qué pasa si hay problemas)
  Semana C: Mantenimiento (cómo cuidar, cada cuánto, qué evitar)
  Semana D: Comparación sin mencionar competidores ("Lo que distingue un trabajo premium")

ESTRUCTURA DEL CARRUSEL EDUCATIVO:
  Slide 1: Headline sorpresa + imagen de soporte (score ≥75)
  Slide 2-5: Un punto por slide (20 palabras máximo por slide)
  Slide 6: Conclusión práctica + CTA implícito
  Slide 7 (opcional): Número de teléfono en dorado + logo

COPY: voz declarativa, un dato concreto por slide, sin "qualité"/"meilleur"

TIEMPO DE PRODUCCIÓN: 25–40 minutos

FORMATO DE LA ORDEN:
  TEMA: ________________
  TIPO: CARRUSEL / IMAGEN ÚNICA
  TITULAR: ________________ (≤8 palabras)
  SLIDES (si carrusel): [una línea por slide]
  IMAGEN PRINCIPAL: [archivo] SCORE: ___ / 100
  CAPTION: ________________
  PLATAFORMAS: Instagram / Facebook / Google Business
  ESTADO: PENDIENTE / EN PRODUCCIÓN / REVISIÓN HUMANA / APROBADO / PROGRAMADO
```

### PIEZA 5 — REMARKETING (retargeting)

```
PROPÓSITO: Reconvertir visitantes del sitio web, leads fríos, y audiencias similares
CANAL: Meta Ads (Facebook/Instagram) — campaña de retargeting
FORMATO: imagen 1:1 o 4:5 para feed; 9:16 para stories
FLOAT V2: AGENTE_MARKETING (05_MARKETING) + MAGAZINE_EDITORIAL_STANDARD

TIPOS DE PIEZA DE REMARKETING:
  A. Testimonio visual → para quien visitó el sitio pero no convirtió
  B. Urgencia elegante → "L'agenda de juillet se remplit." para leads fríos de >7 días
  C. Oferta directa → "Soumission gratuite en 24h" para leads calientes
  D. Antes/Después → para audiencias similares (lookalike) frías

REGLAS ESPECÍFICAS:
  ✓ VISUAL_SCORE ≥ 90/100 si es imagen hero
  ✓ Copy con dato concreto (no genérico)
  ✓ CTA único y directo ("Appelez: 514-266-2504")
  ✓ Número en dorado #C8A45A siempre visible
  ✗ Sin múltiples CTAs compitiendo
  ✗ Sin discursos de marca — solo solución + prueba + acción

TIEMPO DE PRODUCCIÓN: 15–25 minutos

FORMATO DE LA ORDEN:
  TIPO: A / B / C / D
  AUDIENCIA OBJETIVO: [visitantes web / leads >7d / lookalike]
  ASSET: [archivo] SCORE: ___ / 100
  HEADLINE: ________________ (≤8 palabras)
  TEXTO PRINCIPAL: ________________ (≤40 palabras)
  DESCRIPCIÓN: ________________ (≤20 palabras)
  BOTÓN: Obtenir un devis / Appeler maintenant / En savoir plus
  PRESUPUESTO SUGERIDO: ___ CAD/día × ___ días
  ESTADO: PENDIENTE / EN PRODUCCIÓN / REVISIÓN HUMANA / APROBADO / PROGRAMADO
```

## REGLAS DE GESTIÓN DE LA COLA

```
REGLA 1 — Tiempo máximo de producción diaria
  Total de las 5 piezas: máximo 3 horas
  Si una pieza toma más del doble del tiempo estimado → escalar a Daniel

REGLA 2 — Material insuficiente
  Si no hay material ≥90/100 para el Hero → documentar, usar pieza educativa como hero del día
  Si no hay ningún material nuevo → usar material del archivo con nuevo copy
  Si 3 días seguidos sin material → alerta: "SOLICITAR SESIÓN FOTOGRÁFICA urgente"

REGLA 3 — Aprobación humana obligatoria
  Piezas HERO y REEL → requieren revisión de Daniel antes de publicar
  Piezas STORY y EDUCATIVO → pueden publicarse con aprobación del agente
  Piezas REMARKETING → requieren revisión de Daniel siempre (gasto de presupuesto)

REGLA 4 — Prioridad de publicación
  Si hay que elegir entre piezas por falta de tiempo:
  Reel > Hero Post > Story > Educativo > Remarketing

REGLA 5 — No publicar vacíos
  Si una pieza no llega al estándar → no publicar.
  Es mejor no publicar que publicar algo que daña la marca.
```
