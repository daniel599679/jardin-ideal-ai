## CARRUSEL #3 — "L'investissement qui se voit" (Ángulo C)
**Thème:** ROI immobilier — pour propriétaires / investisseurs  
**CPL visé:** <40 CAD (audience plus niche mais conversion plus élevée)

### Structure des slides

| Slide | Contenu | Format |
|-------|---------|--------|
| **1 — Hook chiffre** | Fond blanc. "Votre cour représente 25% de la superficie de votre propriété." | Texte pur — arrête le scroll par contraste |
| **2 — Problème** | `avant_travaux.jpg` + texte: "25% de votre actif immobilier. Inutilisé." | Image avant + chiffre |
| **3 — Solution** | `A7408605.jpg` + texte: "Transformé en espace de vie et de valeur." | Image après — wow |
| **4 — Données** | Infographie (fond clair): "Cour aménagée = +10 à 20% valeur perçue / Temps de vente réduit / Rendement locatif amélioré" | Données / crédibilité |
| **5 — Portfolio** | `A7402004.jpg` + `A7408826.jpg` (split) | Preuve visuelle multi-projets |
| **6 — CTA** | "Calculez le ROI de votre projet." + logo + lien | Conversion |

### Copy complet slide 1
```
Votre cour arrière représente 25% de la superficie de votre propriété.

La plupart des propriétaires l'ignorent complètement.

Faites défiler pour voir ce que ça vaut vraiment. →

Groupe Ideal — Aménagement extérieur à valeur ajoutée
Montréal et banlieues — groupe-ideal.com
```

### Notes de production
- Slide 4 (infographie) : garder les données simples — 3 points max. Police claire, fond blanc ou gris très clair
- Ce carrusel est le plus long à produire — prioriser après Carrusel #1 et #2
- **Audience cible:** Propriétaires 45–65 ans, ayant 2+ propriétés ou déclarés comme investisseurs immobiliers (ciblage Meta)
- **Placement recommandé:** Facebook uniquement (LinkedIn si possible) — Instagram audience trop jeune pour ce message

## DIRECTIVES GÉNÉRALES — PRODUCTION CARRUSELS

### Spécifications techniques
| Paramètre | Valeur |
|-----------|--------|
| Dimensions | 1080×1080 px (carré prioritaire) |
| Format export | JPG (qualité 95%) ou PNG |
| Nombre de slides | 4–6 (jamais plus de 7 — fatigue scroll) |
| Texte slide 1 | ≤8 mots sur l'image |
| Logo | Slide 1 (petit, coin inférieur droit) + slide finale |
