**FLOAT V2 Premium — Reglas Globales de Contenido**
**Versión:** 1.0 | **Fecha:** 2026-06-23
**Alcance:** Aplica a TODO el contenido producido por Jardín Ideal, sin excepción.
**Autoridad:** Este documento tiene autoridad sobre cualquier instrucción anterior de cualquier agente.

## REGLA #1 — VISUAL SCORE GATE (la más importante)

```
NINGÚN CONTENIDO PUEDE EXPORTARSE, PUBLICARSE NI USARSE EN ADS
SI SU VISUAL_SCORE ES MENOR A 90/100.

Sin excepciones. Sin urgencias que justifiquen bajar el estándar.
```

### Escala de uso por score:

| Score | Canal permitido | Uso prohibido |
|-------|----------------|---------------|
| 95–100 | Hero principal, Meta Ads, reel, feed permanente | — |
| 90–94 | Todos los canales normales | — |
| 75–89 | Stories efímeras, carrusel secundario (no primer slide) | Hero, Meta Ads principal, reel |
| 60–74 | Uso interno únicamente (referencia de proyecto) | Cualquier publicación |
| < 60 | Descartar | Todo |

### Cómo medir el score:
Usar el sistema de 5 bloques en `03_PREMIUM_VISUAL_SCORE.md`:
- Composición: 25 pts
- Luz: 25 pts
- Color y retoque: 20 pts
- Limpieza técnica: 15 pts
- Impacto emocional: 15 pts
- **Total: 100 pts**

## REGLA #2 — HORA DE CAPTURA (fotografía)

```
VENTANAS DE LUZ VÁLIDAS (en orden de preferencia):
  1. Hora dorada mañana:  06:30–09:00
  2. Hora azul:           06:00–06:30 o 20:00–20:30 (luz difusa, tonos fríos)
  3. Hora dorada tarde:   16:30–19:00 (mayo–agosto: hasta 20:30)
  4. Día nublado difuso:  cualquier hora — nubosidad homogénea, sin cielo azul intenso

PROHIBIDO FOTOGRAFIAR:
  ✗ Entre 11:00 AM y 14:00 PM (mediodía) → sombras duras, colores quemados
  ✗ Con flash externo como fuente principal → luz plana, aspecto inmobiliario barato
  ✗ Con lluvia o post-lluvia sin secado → reflejos no controlados
```

## REGLA #3 — LIMPIEZA DEL SET

```
ANTES DE FOTOGRAFIAR O FILMAR, verificar:
  ✓ Sin herramientas visibles (palas, carretillas, mangueras)
  ✓ Sin vehículos de trabajo en el frame
  ✓ Sin personas (trabajadores o clientes) salvo consentimiento explícito firmado
  ✓ Sin materiales sobrantes, bolsas, contenedores
  ✓ Sin huellas de trabajo reciente en zonas adyacentes
  ✓ Sin basura del vecindario en el frame

PREPARACIÓN MÍNIMA DEL SET: 10 minutos
  Ver protocolo completo en: 01_HERO_IMAGE_FACTORY.md — "Protocolo de preparación del set"
```

## REGLA #4 — ESTÁNDAR TIPOGRÁFICO (obligatorio en toda pieza con texto)

```
TIPOGRAFÍA APROBADA:
  Headlines:   Playfair Display Bold Italic
  Subtítulos:  Montserrat SemiBold UPPERCASE (tracking +50)
  Body/copy:   Montserrat Regular
  CTA:         Montserrat Bold
  Logo:        posición fija esquina inferior derecha

TIPOGRAFÍA PROHIBIDA:
  ✗ Arial, Helvetica, Roboto → genéricas, no comunican lujo
  ✗ Cualquier fuente decorativa no aprobada (scripts cursivos, display novelty)
  ✗ Texto en negrita genérica sin jerarquía clara
```

## REGLA #5 — PALETA DE COLORES (no negociable)

```
COLORES APROBADOS:
  #052B16   Verde oscuro → fondos, texto principal
  #008E3F   Verde claro  → botones CTA, acentos
  #C8A45A   Dorado       → SOLO para número de teléfono y CTA de conversión directa
  #F5F2EB   Crema        → fondos alternativos, espacios de descanso
  #FFFFFF   Blanco       → texto sobre imagen oscura, espacio negativo

REGLA DEL DORADO:
  El dorado #C8A45A aparece ÚNICAMENTE en el número de teléfono y en el botón CTA.
  Si el dorado aparece en otro elemento → rechazar la pieza.

COLORES PROHIBIDOS:
  ✗ Cualquier color fuera de la paleta aprobada
  ✗ Gradientes de color que mezclen los aprobados con otros
  ✗ Filtros de Instagram o efectos de color no provenientes del proceso Lightroom
```

## REGLA #6 — ESTÁNDAR DE COPY (texto)

```
LA VOZ DE JARDÍN IDEAL:
  ✓ Declarativa: "Cette cour a changé leur vie." — no interrogativa
  ✓ Concreta:    "287 projets réalisés." — no "de nombreux projets"
  ✓ Austera:     "Jardín Ideal. Laval · Westmount." — no "les meilleurs paysagistes"
  ✓ Aspiracional: "La cour que vous méritez." — no "appelez-nous MAINTENANT!!!"

PALABRAS PROHIBIDAS (sin dato concreto que las respalde):
  ✗ "meilleur" / "le meilleur"
  ✗ "qualité" / "de qualité"
  ✗ "professionnel" / "professionnels"
  ✗ "expert" / "experts"
  ✗ "garantie" (sin especificar qué garantiza y por cuánto tiempo)
  ✗ Signos de exclamación (!) en headlines

LONGITUD DEL HOOK:
  Máximo 8 palabras. Si necesita 9, reescribir.
```

## REGLA #7 — MÚSICA (reels y videos)

```
FUENTES APROBADAS (con licencia):
  ✓ Epidemic Sound (suscripción activa requerida)
  ✓ Artlist.io (suscripción activa requerida)
  ✓ Musicbed (suscripción o compra por proyecto)
  ✓ Soundstripe (alternativa)

FUENTES PROHIBIDAS:
  ✗ YouTube Audio Library genérica → reconocible, comunica bajo presupuesto
  ✗ Canciones populares de Spotify/Apple Music sin licencia → riesgo legal + silencio en Instagram
  ✗ Música de iMovie o CapCut preinstalada → genérica
  ✗ Reggaeton, rock pesado → inconsistente con posicionamiento premium

ESTÁNDAR TÉCNICO:
  Nivel de mezcla: -14 LUFS (estándar Instagram)
  Fade in: ≥ 1 segundo (nunca corte seco de entrada)
  Fade out CTA: 1.5 segundos gradual

SELECCIÓN POR EMOCIÓN (ver 04_MUSIC_LIBRARY_SYSTEM.md):
  Transformación → Biblioteca REVELACIÓN (70–90 BPM, piano + cuerdas)
  Awareness ads → Biblioteca ASPIRACIÓN (80–100 BPM, acústico)
  Proceso/craft → Biblioteca CONFIANZA (100–120 BPM, percusión suave)
  CTA directo → Biblioteca URGENCIA ELEGANTE (110–130 BPM)
  Verano/familia → Biblioteca NOSTALGIA Y HOGAR (60–80 BPM)
```

## REGLA #8 — ESTRUCTURA DEL REEL

```
ARQUITECTURA OBLIGATORIA (ver 02_EMOTIONAL_REEL_ENGINE.md):
  0–3s   HOOK         → imagen o texto que detiene el scroll
  3–6s   PROBLEMA     → contexto del antes o del problema
  6–14s  REVELACIÓN   → 80% del presupuesto de producción aquí
  14–18s PRUEBA SOCIAL → dato concreto o testimonio implícito
  18–20s CTA          → número en dorado, una sola acción

MOVIMIENTOS APROBADOS:
  ✓ Slider lento (parallax)
  ✓ Zoom Ken Burns (3-5%)
  ✓ Tilt up o tilt down suave
  ✓ Travelling hacia la propiedad
  ✓ Reveal vertical (de arriba hacia abajo)

MOVIMIENTOS PROHIBIDOS:
  ✗ Handheld shakycam → aspecto amateur
  ✗ Zoom digital rápido → pixelación y aspecto barato
  ✗ Rotación de cámara (dutch tilt) → sin propósito estético
  ✗ Cortes en flash excesivos (más de 3 en 5 segundos)
```

---

## REGLA #9 — COMPOSICIÓN Y ESPACIO

```
PROPORCIONES OBLIGATORIAS:
  Espacio negativo en composición: 30–50% del frame
  En ads premium: hasta 50% vacío si la imagen es suficientemente fuerte
  Una imagen dominante por pieza: no 3 fotos de igual peso compitiendo

REGLA DE HORIZONTE:
  Horizonte perfectamente horizontal (±0.5°)
  Horizonte inclinado >2° → descalificación automática (techo 60/100)

ELEMENTOS PROHIBIDOS EN EL FRAME:
  ✗ Personas identificables sin consentimiento firmado
  ✗ Herramientas de trabajo
  ✗ Vehículos no relacionados con el proyecto
  ✗ Marcas o logos de terceros (Techo-Bloc puede aparecer solo si es material del proyecto)
```
