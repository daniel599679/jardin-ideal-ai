# MAGAZINE EDITORIAL STANDARD
**FLOAT V2 Premium — Módulo 5**
**Versión:** 1.0 | **Fecha:** 2026-06-23
**Estándar:** Architectural Digest × Luxe Interiors × Restoration Hardware Catalog
**Regla absoluta:** Todo lo que publique Jardín Ideal debe poder aparecer, sin modificaciones, en la página de una revista de lujo.

---

## QUÉ SIGNIFICA "ESTÁNDAR EDITORIAL"

Architectural Digest no publica fotos. Publica **declaraciones visuales**.

Un artículo de AD tiene:
- 1 imagen dominante que llena la página — no 6 fotos mediocres
- Texto que no explica la foto — la extiende
- Espacio negativo intencional — el vacío comunica lujo
- Jerarquía tipográfica clara — el lector sabe exactamente dónde mirar

Jardín Ideal aplica estos mismos principios a Instagram, Meta Ads y cualquier pieza impresa.

## PRINCIPIOS EDITORIALES FUNDAMENTALES

### Principio 1 — Menos es más (siempre)
```
Luxe Magazine lo hace: una foto doble página, 12 palabras de headline, 40 de body. Eso es todo.
Jardín Ideal lo hace: 1 hero image, 1 hook de 8 palabras, 1 CTA. Eso es todo.

Por cada elemento que agregas a una pieza, el impacto de cada elemento disminuye.
5 fotos en un ad = ninguna imagen memorable.
1 foto perfecta en un ad = la gente detiene el scroll.
```

### Principio 2 — El espacio negativo es un elemento activo
```
Restoration Hardware no llena sus catálogos. Los respira.
El espacio entre el texto y la imagen no es espacio perdido.
Es el espacio que hace que el ojo descanse sobre lo importante.

Regla Jardín Ideal:
  En cualquier composición (ad, post, carrusel): 30–40% debe ser espacio vacío.
  En ads premium: hasta 50% de espacio vacío si la imagen es suficientemente fuerte.
```

### Principio 3 — La tipografía habla antes de que el texto se lea
```
Cuando alguien ve un ad de Jardín Ideal desde 1 metro de distancia en un teléfono:
  Primero percibe la imagen
  Luego percibe el color del texto y su tamaño
  Luego lee el headline
  Luego (si quiere) lee el subtexto

La tipografía debe funcionar en ese orden.
Playfair Display Bold Italic en el headline = "esto es de una empresa premium"
Montserrat SemiBold en el CTA = "esto es lo que tienes que hacer"
```

### Principio 4 — Una pieza, una idea
```
AD nunca mezcla dos historias en un artículo.
Un reel de Jardín Ideal = una transformación. No tres.
Un carrusel = un proyecto. No cuatro proyectos diferentes.
Un ad = un servicio. No "todos nuestros servicios".

Si tienes muchas cosas que mostrar → hacer múltiples piezas.
No comprimir todo en una.
```

## SISTEMA TIPOGRÁFICO EDITORIAL

### Jerarquía completa

```
NIVEL 1 — EL HOOK (Playfair Display Bold Italic)
  Uso: headline principal, la primera línea que se lee
  Tamaño en imagen: 52–72pt equivalente a 1080px de ancho
  Color: Blanco (#FFFFFF) sobre imagen, o Verde oscuro (#052B16) sobre crema
  Tracking: -10 a -20 (levemente comprimido — look editorial)
  Líneas: máximo 2 líneas. Si necesitas 3, reescribir más corto.
  Ejemplo: "Une cour. Une vie différente."

NIVEL 2 — EL SUBTÍTULO / SERVICIO (Montserrat SemiBold)
  Uso: tipo de servicio, zona geográfica, dato clave
  Tamaño: 18–24pt
  Color: igual al hook pero con opacidad 70%, o en dorado (#C8A45A) solo si es CTA
  Tracking: +50 (espaciado generoso — elegante)
  Transform: UPPERCASE siempre para este nivel
  Ejemplo: "PAVÉ-UNI · WESTMOUNT · 2026"

NIVEL 3 — EL BODY / DESCRIPCIÓN (Montserrat Regular)
  Uso: descripción de 1–3 líneas, solo si el formato lo permite
  Tamaño: 14–16pt
  Color: Blanco o #052B16 según fondo
  Máximo: 40 palabras. Si necesitas más, la pieza es un artículo, no un ad.
  Solo usar en formatos >1080px o en carrusel (donde hay espacio)

NIVEL 4 — EL CTA (Montserrat Bold)
  Uso: llamado a la acción y número de teléfono
  Tamaño: 18–22pt
  Color: #C8A45A (dorado) SIEMPRE para el número de teléfono
  El botón: fondo #008E3F (verde claro) con texto blanco
  Ejemplo: "Appelez: 514-266-2504"

NIVEL 5 — EL LOGO (posición fija)
  Posición: esquina inferior derecha SIEMPRE
  Tamaño: 8–10% del ancho de la imagen
  Versión: logo blanco sobre imágenes oscuras / logo verde sobre fondo claro
  Nunca centrado en la composición final
```

## SISTEMA DE LAYOUT — 4 TEMPLATES EDITORIALES

### TEMPLATE A — "La Portada" (hero image dominant)
```
┌─────────────────────────────────────┐
│                                     │
│          IMAGEN HERO                │
│         (85% del frame)             │
│                                     │
│                                     │
│                                     │
│                                     │
├─────────────────────────────────────┤
│  HOOK HEADLINE (Playfair)           │
│  SUBTÍTULO (Montserrat SemiBold)    │
│                          [LOGO]     │
└─────────────────────────────────────┘

Uso: Feed posts, hero de carrusel, Meta Ads awareness
Proporción imagen:texto = 85:15
El texto está FUERA de la imagen, no encima
Estilo: Architectural Digest, Luxe Magazine
```

### TEMPLATE B — "El Overlay Elegante" (texto sobre imagen)
```
┌─────────────────────────────────────┐
│                                     │
│          IMAGEN HERO                │
│                                     │
│                                     │
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓      │
│  ▓  HOOK (Playfair, blanco)   ▓      │
│  ▓  SUBTÍTULO (Montserrat)    ▓      │
│  ▓  CTA  [514-266-2504]  [LOGO] ▓   │
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓      │
└─────────────────────────────────────┘

Overlay: gradiente negro de 0% (arriba) a 60% (abajo) — NO caja de color sólida
El gradiente debe empezar a ⅔ de la imagen — la imagen respira arriba
Uso: Instagram Reels thumbnail, Stories, Meta Ads con texto integrado
```

### TEMPLATE C — "El Split Editorial" (para carrusel y before/after)
```
┌─────────────────┬───────────────────┐
│                 │                   │
│  IMAGEN 1       │  IMAGEN 2         │
│  (AVANT)        │  (APRÈS)          │
│                 │                   │
│                 │                   │
├─────────────────┴───────────────────┤
│  HOOK                [LOGO]         │
└─────────────────────────────────────┘

Línea divisoria: 2px, blanco o dorado — nunca más gruesa
Etiquetas: "AVANT" y "APRÈS" en Montserrat SemiBold 12pt uppercase
Posición etiquetas: esquina superior del panel respectivo
Uso: primer slide del carrusel, Meta Ads before/after
```

### TEMPLATE D — "El Carrusel Editorial" (slides 2–7)
```
SLIDE ESTRUCTURA:
┌─────────────────────────────────────┐
│  [NÚMERO] ──────────────────────    │  ← Numeración: "02 / 06" Montserrat 12pt
│                                     │
│          IMAGEN O DETALLE           │
│                                     │
│                                     │
│  DESCRIPCIÓN BREVE                  │
│  (máximo 2 líneas, Montserrat)      │
│                               [▶]   │  ← Flecha discreta: "swipe" indicator
└─────────────────────────────────────┘

Progresión del carrusel:
  Slide 1: Hero principal (Template A o C)
  Slides 2-4: Detalles del proyecto (textura, materiales, ángulos)
  Slide 5: Proceso (antes/durante si hay fotos)
  Slide 6: Dato de impacto o prueba social
  Slide 7 (último): CTA completo con número de teléfono
```

## COPY EDITORIAL — LA VOZ DE JARDÍN IDEAL

### Principios de la voz

```
DECLARATIVO, nunca interrogativo
  ✗ "Vous voulez une belle cour?" — suena a vendedor de teléfono
  ✓ "Une cour transformée change votre rapport à votre maison." — declaración

CONCRETO, nunca vago
  ✗ "Nous créons de beaux espaces extérieurs de qualité"
  ✓ "287 projets réalisés à Montréal. 0 client insatisfait."

AUSTERO, nunca hiperbólico
  ✗ "Les meilleurs paysagistes du Québec!!!"
  ✓ "Jardín Ideal. Laval · Westmount · TMR."
  (el nombre de las zonas premium comunica el posicionamiento sin decirlo)

ASPIRACIONAL, nunca desesperado
  ✗ "Appelez-nous MAINTENANT pour une soumission GRATUITE!!!"
  ✓ "L'agenda de juillet se remplit. 514-266-2504."
```

### Banco de headlines editoriales por emoción

```
REVELACIÓN
  "Avant : un terrain. Après : une vie."
  "Ce n'est pas un patio. C'est où votre vie se passe."
  "Trois semaines. Une transformation permanente."

ASPIRACIÓN
  "La cour que vous méritez."
  "Votre propriété, à sa juste valeur."
  "Westmount a un nouveau standard."

CONFIANZA
  "Pavé-Uni, Béton, Pierre. Installé pour durer 40 ans."
  "Chaque joint, chaque pierre, chaque détail."
  "[X] familles à Montréal nous ont fait confiance."

URGENCIA ELEGANTE
  "L'été 2026 est presque complet."
  "Nous acceptons 4 nouveaux projets ce mois-ci."
  "Soumission gratuite : 3 jours pour décider."

HOGAR
  "Le soir, vous méritez un espace qui vous ressemble."
  "Votre famille mérite la meilleure version de votre maison."
  "C'est chez vous. Ça doit être parfait."
```

### Longitud del copy por formato

```
INSTAGRAM FEED POST:
  Caption: 80–150 palabras
  Estructura: Hook 1 línea → 2-3 párrafos → CTA
  Hashtags: 8–12, en el primer comentario (no en el caption)

META ADS (awareness):
  Headline: máximo 8 palabras
  Primary text: 1–2 oraciones (25–40 palabras)
  Description: 1 oración (15–20 palabras)

META ADS (conversión/retargeting):
  Headline: 8 palabras con dato específico
  Primary text: 3–5 oraciones con prueba social y CTA
  Description: CTA directo

CARRUSEL — SLIDE COPY:
  Máximo 20 palabras por slide
  Cada slide = un concepto, no un párrafo

STORIES:
  Máximo 5 palabras de texto sobre imagen
  El visual comunica el 90%
```

## CHECKLIST EDITORIAL PRE-PUBLICACIÓN

```
IDENTIDAD VISUAL
☐ Tipografía: solo Playfair Display Italic (headlines) + Montserrat (body/CTA)
☐ Paleta de colores: solo #052B16 / #008E3F / #C8A45A / #F5F2EB / #FFFFFF
☐ Dorado #C8A45A: solo en CTA de teléfono — no en ningún otro elemento
☐ Logo en esquina inferior derecha, tamaño correcto

COMPOSICIÓN
☐ 30–50% de espacio negativo en la composición
☐ Una imagen dominante — no múltiples de igual peso
☐ Texto no superpuesto sobre el área de más detalle de la imagen
☐ Jerarquía clara: headline > servicio > CTA

COPY
☐ Headline: máximo 8 palabras
☐ Voz: declarativa, austera, aspiracional
☐ Número de teléfono en dorado: 514-266-2504
☐ Sin signos de exclamación en headline
☐ Sin "meilleur", "qualité", "professionnel" sin dato concreto que lo respalde

TÉCNICO
☐ Resolución mínima: 1080 × 1080 px
☐ Texto legible a tamaño thumbnail (400px de vista previa)
☐ Score Visual mínimo: 90/100 (Archivo 03)
☐ No usar filtros de Instagram — el color grade viene de Lightroom

LEGAL
☐ Música con licencia (Epidemic Sound / Artlist / Musicbed)
☐ No usar imágenes de terceros sin permiso explícito
☐ Si hay personas visibles: consentimiento del cliente firmado
```
