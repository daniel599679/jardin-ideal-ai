# REPORTE MENSUAL — PLANTILLA
**Llenar el primer lunes de cada mes | Tiempo estimado: 60 minutos**
**Fuentes: 4–5 reportes semanales + Meta Ads + Google Ads + CRM + Operaciones**

---

## CÓMO USAR ESTA PLANTILLA
1. Copiar a `reporte_mes_YYYY-MM.md` (ej: `reporte_mes_2026-06.md`)
2. Consolidar datos de los reportes semanales del mes
3. Comparar con el mes anterior Y con el objetivo anual
4. Este reporte es la base para decisiones estratégicas: presupuestos, contrataciones, pausas de campañas
