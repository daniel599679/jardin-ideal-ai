# CRM — PLANTILLAS DE MENSAJES
**Todos los mensajes están en francés québécois | Uso interno en español**

> Copiar, adaptar el nombre y servicio, enviar. No improvisar cuando hay una plantilla disponible.

## BLOQUE 1 — PRIMER CONTACTO

### 1.1 — Primer contacto por WhatsApp (lead de Meta Ads)
```
Bonjour [Prénom] !

Merci de votre intérêt pour Jardín Ideal. 😊

Je suis [Nom], et je serais ravi(e) de discuter de votre projet
de [service ex: pavé-uni / aménagement paysager].

Seriez-vous disponible pour un court appel aujourd'hui ou demain
afin que je puisse mieux comprendre vos besoins ?

Au plaisir,
[Nom] | Jardín Ideal
📞 [Número]
```

### 1.2 — Primer contacto por llamada (guion)
```
"Bonjour, est-ce que je parle bien à [Prénom] ?
Parfait ! Je m'appelle [Nom], je vous appelle de chez Jardín Ideal.
Vous avez fait une demande d'information pour [service].
Est-ce que c'est un bon moment pour parler 2-3 minutes ?"

→ Si dice sí: pasar a preguntas de calificación
→ Si dice no: "Pas de problème ! Quand serait-il plus commode
   pour vous que je vous rappelle ?"
```

### 1.3 — Primer contacto por DM (Instagram/Facebook)
```
Bonjour [Prénom] !

Merci pour votre message. 🌿

Nous serions heureux de vous aider avec votre projet.
Pour mieux vous orienter, pourriez-vous nous dire :

1. De quel type de travaux s'agit-il ?
2. Dans quelle ville est située votre propriété ?

On vous répond rapidement !

Jardín Ideal 🍃
```

## Reglas de uso
- Siempre personalizar `[Prénom]`, `[service]`, `[Nom]`, `[Número]`, fechas y montos
- Nunca enviar una plantilla sin revisar que todos los campos `[...]` estén reemplazados
- En llamadas: el guion es una guía, no un script rígido — adaptar al tono del cliente
- Tono general: profesional, cálido, directo — nunca presionar ni ser insistente
