# CRM — CALIFICACIÓN DE LEADS
**Sistema de puntuación para priorizar tiempo y recursos**

> No todos los leads valen lo mismo. Este sistema permite saber en 5 minutos si un lead merece visita técnica inmediata, seguimiento normal, o no es viable.

---

## Sistema de Puntuación (0–100 puntos)

### DIMENSIÓN 1 — Servicio solicitado (0–25 pts)

| Servicio | Puntos | Razón |
|---|---|---|
| Pavé-uni completo (entrée + cour) | 25 | Ticket alto, alta frecuencia |
| Entrée seule (pavé-uni) | 20 | Ticket medio-alto, rápido de ejecutar |
| Cour arrière completa | 22 | Ticket alto, proyecto de impacto |
| Cour avant | 18 | Ticket medio |
| Piscina | 25 | Ticket más alto, temporada específica |
| Escaliers extérieurs | 15 | Ticket medio |
| Allée latérale | 12 | Ticket bajo-medio |
| Renovación exterior/interior | 20 | Variable, potencial alto |
| Servicio vago / "quiero un presupuesto" | 5 | Baja intención, alta comparación |

---

### DIMENSIÓN 2 — Zona y desplazamiento (0–20 pts)

| Zona | Puntos |
|---|---|
| Montreal (isla) — zona central | 20 |
| Laval | 20 |
| Rive-Sud (Longueuil, Brossard, St-Lambert) | 18 |
| Rive-Nord (Blainville, Terrebonne, St-Jérôme) | 15 |
| Fuera de radio habitual (<60 min) | 10 |
| Fuera de radio habitual (>60 min) | 0 — no viable |

---

### DIMENSIÓN 3 — Presupuesto aproximado (0–25 pts)

| Rango declarado por el cliente | Puntos |
|---|---|
| No lo menciona / "no sé" | 10 (neutro) |
| "Quiero lo más barato posible" | 3 |
| "Tengo un presupuesto de $X" (razonable) | 20 |
| "No tengo límite de presupuesto" | 25 |
| Presupuesto muy bajo para el servicio pedido | 2 |

**Presupuestos mínimos de referencia por servicio:**
| Servicio | Mínimo viable |
|---|---|
| Entrée pavé-uni | ~8,000 CAD |
| Cour avant complète | ~12,000 CAD |
| Cour arrière + terrasse | ~20,000 CAD |
| Piscina creusée | ~50,000 CAD |
| Escaliers extérieurs | ~5,000 CAD |

---

### DIMENSIÓN 4 — Urgencia y timing (0–20 pts)

| Situación | Puntos |
|---|---|
| "Para esta primavera / lo antes posible" | 20 |
| "Para este verano" | 15 |
| "Para el otoño" | 10 |
| "Para el año que viene" | 5 |

| "Solo quiero comparar precios" | 2 |

---

### DIMENSIÓN 5 — Señales de calidad del lead (0–10 pts)
*(Evaluación subjetiva durante el primer contacto)*

| Señal | Puntos |
|---|---|
| Da información detallada sin que se le pida | +3 |
| Ya tiene fotos o planos del espacio | +3 |
| Fue referido por un cliente actual | +3 |
| Tiene fecha concreta para el proyecto | +2 |
| Muestra urgencia genuina (no presión artificial) | +2 |
| Ya rechazó cotizaciones anteriores "muy caras" | -3 |
| Pide descuentos antes de ver la propuesta | -3 |
| Historia de proyectos abandonados | -3 |

## Clasificación Final

| Puntaje total | Categoría | Acción |
|---|---|---|
| 75–100 | ⭐⭐⭐ **Lead Premium** | Contactar en <30 min, priorizar visita esta semana |
| 50–74 | ⭐⭐ **Lead Calificado** | Contactar en <2h, visita en 5–7 días |
| 25–49 | ⭐ **Lead Frío** | Seguimiento normal, no priorizar sobre los anteriores |
| 0–24 | ❌ **No viable** | Descartar amablemente o archivar |

---

## Calificación Rápida (primeros 5 minutos)

Si no hay tiempo para el sistema completo, hacer estas 4 preguntas:

```
1. ¿Qué trabajo quiere hacer exactamente?
   → ¿Es un servicio que ofrecemos? ¿Ticket razonable?

2. ¿Dónde está ubicada la propiedad?
   → ¿Está en nuestra zona de servicio?

3. ¿Cuándo quiere empezar?
   → ¿Es este año? ¿Esta temporada?

4. ¿Tiene una idea del presupuesto?
   → ¿Es realista para el servicio pedido?
```

**Si las 4 respuestas son positivas → Lead calificado, agendar visita.**
**Si 2 o más son negativas → evaluar si vale la pena continuar.**
